# Menu1211 (1.21.1)

- `/메뉴` : 메뉴 열기  
- `/메뉴 리로드` : 설정 리로드  
- `/메뉴 아이템` : 손에 든 아이템을 메뉴 오프너로 설정 (오른쪽 클릭하면 메뉴 열림)  
- 웅크린 상태에서 `F`(손바꾸기) 키 → 메뉴 열기 (config로 on/off)

## 왜 예전엔 빈 GUI만 떴나요?
- 1.21.1에선 인벤토리 **제목 비교**만으로 판별하면 종종 실패합니다. 커스텀 `InventoryHolder`를 도입해 확실하게 메뉴만 막습니다.
- 잘못된 `Material` 이름, 비어있는 `items` 섹션 등도 빈칸의 원인입니다. 로드 시 경고를 출력하고 정상 항목만 채우도록 방어 코드를 넣었습니다.

## 설정
`src/main/resources/config.yml` 참고 (`&` 색코드, `{player}` 치환 지원).

## 빌드
```bash
mvn -B package
```
결과: `target/Menu1211-*.jar`
